# Contributing to pi-zai-usage

Thanks for your interest! Before opening a pull request, please ensure:

- [ ] All tests pass: `npm test`
- [ ] Linting passes: `npm run lint`
- [ ] Type checking passes: `npm run typecheck`
- [ ] Formatting is clean: `npm run format:check`
- [ ] Test coverage stays at 90%+ for statements, branches, functions, and lines

## Development Setup

```bash
npm install
npm test
```

## Making Changes

1. Create a feature branch from `main`
2. Make your changes with tests
3. Run the full verification suite
4. Open a pull request with a clear description
